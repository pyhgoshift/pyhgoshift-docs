## 4. 기술 스택 최신 동향 (웹 조사)

### n8n (자동화 엔진)
2026년 n8n은 400개 이상 통합을 지원하는 시각적 AI 워크플로우 엔진으로 성장했다. LLM 체이닝, 벡터 DB 연동, 도구-사용 에이전트를 코드 없이 구성 가능. 셀프호스팅 시 완전 온프레미스, SSO SAML, 암호화 시크릿 저장소 지원. 1Password Connect Server, AWS Secrets Manager 등 외부 시크릿 프로바이더 연동 추가됨.
> 참고: [n8n Self-Host Guide 2026](https://northflank.com/blog/how-to-self-host-n8n-setup-architecture-and-pricing-guide)

### Whisper STT (음성→텍스트)
2026년 기준 `faster-whisper` 기반 Docker 이미지가 성숙. `hwdsl2/whisper-server`는 OpenAI 호환 API, CUDA GPU 가속, 오프라인 모드 지원. tiny~large-v3-turbo 전 모델 지원. 최소 RAM 700MB 필요. NAS Docker에 직접 설치 가능.
> 참고: [Whisper Docker 2026](https://oneuptime.com/blog/post/2026-02-08-how-to-run-whisper-speech-to-text-in-docker/view)

### Cloudflare Pages (docs 호스팅)
무료 티어에서 무제한 대역폭, 전 세계 CDN, 자동 HTTPS 제공. git push → 1분 이내 전 세계 배포. 롤백, 커밋별 미리보기 URL, Cloudflare Access 연동 지원. Docsify 같은 빌드 불필요 정적 사이트에 최적.
> 참고: [Cloudflare Pages Docs](https://developers.cloudflare.com/pages/)

### Claude API (AI 핵심)
2026년 5월 기준 Claude Opus 4.7 / Sonnet 4.6 / Haiku 4.5 운영 중. Files API, Skills(beta), MCP Connector(beta), Prompt Caching, Citations, Batch Processing 지원. AWS에서 Claude Platform 정식 출시 (Managed Agents, Web Search, Code Execution 포함). Anthropic "Dreaming" 기능: 에이전트 세션 간 메모리 자동 정리·통합.
> 참고: [Claude API 2026](https://platform.claude.com/docs/en/about-claude/models/overview)
