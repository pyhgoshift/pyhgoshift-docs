## 4. 기술 동향 웹 조사

### n8n Telegram 봇 통합 (2026)
n8n은 Telegram Trigger(webhook) + Telegram(액션) 노드 2개로 완전한 봇 구축 가능.
**Webhook 방식**이 Polling보다 빠르고 효율적 — n8n.pyhgoshift.com은 고정 URL이므로 최적 환경.
에러 핸들링: Error Trigger 워크플로우를 추가하면 장애 발생 시 Telegram으로 자동 알림 수신 가능.
> 참고: [n8n Telegram Integration Guide](https://n8nautomation.cloud/blog/n8n-telegram-integration-bot-automations)

### Synology NAS n8n Docker 볼륨 마운트
**핵심 이슈**: n8n 컨테이너가 기본적으로 외부 파일 시스템 접근 차단 (`N8N_BLOCK_FILE_ACCESS_TO_N8N_FILES=true`).
**해결 방법**:
1. Container Manager → n8n → 편집 → 볼륨 탭
2. 호스트 경로: `/volume1/pyhgoshift/AI-Claude/shared/knowledge_base/inbox`
3. 컨테이너 경로: `/pyhgoshift-inbox` (또는 동일 경로)
4. 환경변수 추가: `N8N_BLOCK_FILE_ACCESS_TO_N8N_FILES=false`
> 참고: [Synology n8n Docker 가이드](https://mariushosting.com/how-to-install-n8n-on-your-synology-nas/)
